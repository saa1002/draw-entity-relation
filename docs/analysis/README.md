# Análisis del sistema heredado

## Contexto

Este directorio contiene la documentación de trabajo asociada al análisis del sistema heredado utilizado como base para el Trabajo Fin de Grado (TFG).

El objetivo de esta fase no es generar la memoria final, sino comprender en profundidad la arquitectura, el modelo interno y el funcionamiento del editor de diagramas Entidad–Relación existente, con el fin de justificar y planificar correctamente las mejoras posteriores.

## Estado actual del análisis

En el momento actual, el análisis se encuentra en una fase inicial centrada en:

- Identificación de los componentes principales de la aplicación.
- Comprensión de la arquitectura general del editor.
- Estudio del modelo interno del diagrama y su sincronización con la representación gráfica.
- Análisis del flujo de interacción del usuario y la gestión de eventos.

La documentación técnica detallada se irá incorporando progresivamente a lo largo de esta fase, conforme se consolide la comprensión del sistema y se estructure el contenido de forma adecuada para su posterior inclusión en la memoria del TFG.

## Metodología

El análisis se está realizando de forma incremental, priorizando:

- Claridad conceptual frente a exhaustividad prematura.
- Justificación técnica de las decisiones de diseño existentes.
- Separación entre documentación de trabajo y memoria final.

Una vez finalizada esta fase de análisis, se definirá un primer conjunto de mejoras a implementar sobre el sistema heredado.
